# Research-CryptoAPI

Репозиторий для проверки технических аспектов бесплатных API криптовалют.

## Протестированные сервисы

| Сервис | Эндпоинт | Ключ |
|--------|----------|------|
| CoinGecko | /ping | Не требуется |
| Bybit | /v5/market/tickers | Не требуется |
| CryptoCompare | /data/price | Бесплатный |
